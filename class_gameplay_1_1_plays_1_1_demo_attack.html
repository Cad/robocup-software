Gameplay::Plays::DemoAttack Class Reference

Detailed Description

This play just drives all robots to kick the ball continuously Useful for testing basic operation of the system, and should probably only be used with 1-2 robots on the field, as it will get chaotic.

Uses parameters in the Config GUI to adjust flags
